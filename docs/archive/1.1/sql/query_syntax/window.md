title: WINDOW Clause
---

The `WINDOW` clause allows you to specify named windows that can be used within [window functions]({% link docs/archive/1.1/sql/functions/window_functions.md %}). These are useful when you have multiple window functions, as they allow you to avoid repeating the same window clause.
