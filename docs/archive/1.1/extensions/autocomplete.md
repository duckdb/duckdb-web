The `autocomplete` extension adds supports for autocomplete in the [CLI client]({% link docs/archive/1.1/api/cli/overview.md %}).
The extension is shipped by default with the CLI client.

## Behavior

For the behavior of the `autocomplete` extension, see the [documentation of the CLI client]({% link docs/archive/1.1/api/cli/autocomplete.md %}).

## Functions

| Function                          | Description                                          |
|:----------------------------------|:-----------------------------------------------------|
| `sql_auto_complete(query_string)` | Attempts autocompletion on the given `query_string`. |

## Example

```sql
SELECT *
FROM sql_auto_complete('SEL');
```
