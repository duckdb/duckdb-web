This page contains installation options for an old version of DuckDB. We recommend using the latest stable release.

Binaries are available for major programming languages and platforms. If there are no pre-packaged binaries available, consider building DuckDB from source.

Version

  • 0.7.1 (stable release)
  • Nightly build (bleeding edge)

Environment

  • Command line
  • Python
  • R
  • Java
  • Node.js
  • Rust
  • Go
  • C/C++
  • ODBC

Platform

  • Windows
  • macOS
  • Linux
The package is available for all platforms.

Download method

  • Package manager
  • Direct download

Architecture

  • x86_64
  • arm64
The package supports both architectures.
