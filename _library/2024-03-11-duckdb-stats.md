[DuckDB Stats](https://www.duckdbstats.com/) is a real-time analytics dashboard showing PyPI download statistics for the DuckDB Python package.
The site is built with Next.js, DuckDB, and MotherDuck, and exposes the raw download data via a shared MotherDuck connection string for direct querying.
