  extended_description: |
    Delta Classic lets you attach a directory of Delta Lake tables as a single
    DuckDB database, instead of attaching each table individually. It discovers
    tables by scanning for subdirectories containing a _delta_log folder.

    The extension supports two directory layouts:
    - Single-schema: the attached path directly contains Delta tables
    - Multi-schema: the attached path contains subdirectories, each treated as
      a schema, with Delta tables inside them

    All Delta reading is delegated to the existing Delta extension via internal
    ATTACH ... (TYPE DELTA). Supports PIN_SNAPSHOT and all storage backends
    (local filesystem, S3, ABFSS, GCS).
