  extended_description: |
    [NSV](https://nsv-format.org) is a plain text data format for sequences of sequences.
    A subset of these where the nested sequences all have the same length is of interest, for that is effectively a table.

    The NSV format itself is type-agnostic. This extension attempts to closely follow what the CSV extension does for the type narrowing logic, the column name convention, and the interface surface. A subset of that, of course.

    In terms of expressive power, NSV can be used for anything CSV is used for, but being a format much more amenable to parsing, the performance of that is far higher than for CSV, and so is its readability.
    This extension is intended to enable performant queries against the files using the format.
